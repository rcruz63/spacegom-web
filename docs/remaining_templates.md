# treasury.html

## Overview
`treasury.html` is the financial management interface for the SpaceGOM Companion application. It extends `base.html` and provides comprehensive treasury tracking, transaction recording, and financial oversight.

## Key Features
- **Financial Dashboard**: Current balance, monthly expenses, loans, difficulty level
- **Transaction Recording**: Manual entry of income/expense transactions
- **Transaction History**: Complete financial ledger with categorization
- **Category System**: Trade, missions, supplies, repairs, fuel, salaries, loans, other

## Structure
- **Summary Cards**: Key financial metrics display
- **Transaction Form**: Manual entry with amount, category, description
- **History Table**: Date, description, category, amount with color coding

# missions.html

## Overview
`missions.html` is the mission management interface for the SpaceGOM Companion application. It extends `base.html` and handles campaign objectives and special missions.

## Key Features
- **Mission Dashboard**: Active, completed, failed mission counts
- **Campaign Objectives**: Long-term campaign goal creation and tracking
- **Special Missions**: Book-referenced mission creation
- **Mission Status**: Active/completed/failed state management

## Structure
- **Summary Cards**: Mission statistics overview
- **Creation Forms**: Campaign objectives and special missions
- **Mission List**: Active missions with status and actions
- **Resolution System**: Success/failure handling

# logs.html

## Overview
`logs.html` is the event logging interface for the SpaceGOM Companion application. It extends `base.html` and provides comprehensive game event tracking and history.

## Key Features
- **Event Log**: Chronological game event display
- **Event Types**: Personnel, trade, missions, system events
- **Filtering**: Event type and date range filtering
- **Search**: Text-based event searching
- **Export**: Log data export functionality

## Structure
- **Log Display**: Scrollable event timeline
- **Filter Controls**: Type and date filtering
- **Search Bar**: Text search functionality
- **Export Options**: Data export buttons
